# GitHub 기여 통계 데이터 명세

---

## 0. 기여 저장소 정의

"기여 저장소"는 아래 조건 중 하나 이상을 충족하는 저장소를 의미한다.

| 조건 | 포함 여부 |
|------|----------|
| 커밋 1회 이상 | O |
| PR 1회 이상 제출 | O |
| 이슈 1회 이상 생성 | X |
| 본인 소유 저장소 | O |
| 포크한 저장소 (기여 없음) | X |
| 포크 후 커밋/PR 있음 | O |

---

## 1. 최근 기여활동

**대상**: 모든 공개 저장소

| 데이터 | 설명 | 활용 |
|--------|------|------|
| repository_name | 저장소 이름 | 표시용 |
| description | 프로젝트 설명 | 표시용 |
| stargazers_count | 스타 수 | 통계 표시, 영향성 점수 산출 |
| user_commit_count | 해당 저장소 내 사용자 커밋 수 | 통계 표시, 활동 수준 점수 산출 |
| user_pr_count | 해당 저장소 내 사용자 PR 수 | 통계 표시, 활동 수준 점수 산출 |
| last_commit_date | 최근 커밋 일자 | 표시용 |

---

## 2. 전체 기여 내역

| 데이터 | 설명 | 활용 |
|--------|------|------|
| contributed_repo_count | 기여한 저장소 수 | 통계 표시, 다양성 점수 산출 |
| total_commit_count | 총 커밋 수 | 통계 표시, 비교용 |
| total_additions | 추가한 라인 수 | 통계 표시 |
| total_deletions | 삭제한 라인 수 | 통계 표시 |
| total_issue_count | 생성한 이슈 수 | 통계 표시, 비교용 |
| total_pr_count | 생성한 PR 수 | 통계 표시, 비교용 |

## 3. 기여내역 비교

| 데이터 | 설명 | 활용 |
|--------|------|------|
| avg_commit_count | 평균 커밋 수 | 사용자 수치와 비교 |
| avg_star_count | 평균 스타 수 | 사용자 수치와 비교 |
| avg_pr_count | 평균 PR 수 | 사용자 수치와 비교 |
| avg_issue_count | 평균 이슈 수 | 사용자 수치와 비교 |
| user_commit_count | 나의 커밋 수 | 평균과 비교 표시 |
| user_star_count | 나의 스타 수 (소유 저장소 합계) | 평균과 비교 표시 |
| user_pr_count | 나의 PR 수 | 평균과 비교 표시 |
| user_issue_count | 나의 이슈 수 | 평균과 비교 표시 |

## 4. GitHub 기여점수

### 4.1 활동 수준 (최대 3점)

기여한 저장소 중 가장 높은 점수를 적용한다.

| 점수 | 조건 |
|------|------|
| 3점 | 커밋 100회 이상 AND PR 20회 이상 |
| 2점 | 커밋 30회 이상 AND PR 5회 이상 |
| 1점 | 커밋 5회 이상 OR PR 1회 이상 |
| 0점 | 위 조건 미충족 |

### 4.2 활동 다양성 (최대 1점)

기여한 저장소 개수를 기준으로 산출한다.

| 점수 | 조건 |
|------|------|
| 1.0점 | 기여 저장소 10개 이상 |
| 0.7점 | 기여 저장소 5~9개 |
| 0.4점 | 기여 저장소 2~4개 |
| 0점 | 기여 저장소 1개 이하 |

### 4.3 활동 영향성 (최대 5점, 보너스)

오픈소스 생태계 기여도를 평가하며, 조건 충족 시 가산한다.

| 점수    | 조건                         |
| ----- | -------------------------- |
| +2점   | 본인 소유 저장소 중 스타 100개 이상 보유  |
| +1.5점 | 스타 1,000개 이상 외부 저장소에 PR 병합 |
| +1점   | 본인 PR로 해결된 이슈 10개 이상       |
| +0.5점 | 포크한 저장소에 원본 PR 병합          |
