## 사용 가능한 통계 필드

챌린지 조건에 사용할 수 있는 백엔드 API 필드:

### overall-history
- `contributedRepoCount`: 기여한 레포지토리 수
- `totalCommitCount`: 총 커밋 수
- `totalAdditions`: 총 추가 라인 수
- `totalDeletions`: 총 삭제 라인 수
- `totalIssueCount`: 총 이슈 수
- `totalPrCount`: 총 PR 수

### contribution-comparison
- `userCommitCount`: 사용자 커밋 수
- `userStarCount`: 사용자 스타 수
- `userPrCount`: 사용자 PR 수
- `userIssueCount`: 사용자 이슈 수

### contribution-score
- `activityScore`: 활동 점수
- `diversityScore`: 다양성 점수
- `impactScore`: 영향력 점수
- `totalScore`: 총점
